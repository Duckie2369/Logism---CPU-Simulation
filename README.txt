1. Run the logisim-win-2.7.1.exe
2. File -> Open -> Choose the 'CPU.circ' in 'Circuit' file
3. Double-click on 'CPU' in the top left tab
4. Click 'Registers' tab in the bottom left tab to follow the change in registers
5. You can open the example_set_ins.txt to see the compiled instructions
5. Simulate -> Tick Once or press Ctrl + T to tick the clock
6. Do it multiple times and you can see the results in the 'Registers' tab
